# Сообщения Для обмена сообщениями используется jqeruy.triggerHandler() и jquery.on(), абстрагируемые через Pi.sub и Pi.pub. Для выбора элемента, в который подается сообщение, используются селекторы jquery. Элемент определяет реакцию на сообщения через метод Pi.sub. Подача сообщений при нажатии на "кнопки". Формат сообщений: "selector@message ... selector@message", гдe selector - css-селектор, а message - сообщение в произвольной форме (page/load). Специальный селектор parent эквивалентен вызову $.parent()
