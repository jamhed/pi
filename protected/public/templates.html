# Шаблоны Шаблоны определяются через элемент a/Source. Сам шаблон может как находится внутри элемента, так и быть загружен с сервера. Каждый шаблон имеет имя, по которому применяется элементом типа a/Template. a/Template применяет шаблон с параметрами, которые либо содержатся внутри элемента a/Template, либо загружаются с сервера.
