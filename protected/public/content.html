# Pi Библиотека для сборки сайтов из шаблонов, с трансформациями. Используются amd-модули и require.js, вклюючая некоторый ui: показ диалогов, таблиц с листанием, выбор даты, выбор времени, и поле ввода с подсказкой. ### Архитектура html-элементы помечаются специальным атрибутом pi=[имя]. При загрузке страницы происходит поиск всех таких элементов, и загрузка модулей по имени посредством requirejs. Загруженный модуль, являющийся классом, наследуемым от a/Pi, инстанциируется объектом класса для каждого такого элемента сразу после загрузки. ### ToDo * листание pager с догрузкой содержимого * отложенные сообщения pub/sub при наличии отсутствия html-элемента цели * редактирование html и css-страниц, образующих проект (index.html, style.css) * поле для немедленного редактирования (сохранение при потере фокуса) * check-box для немедленного редактирования * обработка ошибок компиляции (взаимозависимые элементы и события по именам): * отсутствие mSource * отсутствие обработчиков событий pub/sub ### Библиотеки 1.
